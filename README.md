# vaultwares-v1

**Official VaultWares Company Website**  
**Part of the VaultWares Ecosystem** • <a href="https://docs.vaultwares.com">docs.vaultwares.com</a> • <a href="https://vaultwares.com">vaultwares.com</a>

**VaultWares is a cybersecurity and data privacy-focused company that offers products and services built 100% from Free Open-Source Software &amp; Hardware.**

## Overview
This repository hosts the main marketing site, product showcase, documentation links, and company information. It serves as the public entry point for the entire VaultWares ecosystem.

## Features
- Clean, privacy-first design using vault-themes
- Product catalog (Vault Central, Pipelines, Flows, Player, Agentciation, etc.)
- Strong emphasis on 100% FOSS philosophy
- Links to all repositories and central documentation
- Agent integration showcase

## Quick Start (Development)

```bash
git clone https://github.com/p-potvin/vaultwares-v1.git
cd vaultwares-v1
npm install
npm run dev
```
